# Multi Counter App (LazyColumn)

A simple Android app built with Jetpack Compose that displays a scrollable list of independent counters. Each counter can be incremented, decremented, and customized individually.

# Features

Scrollable list of counters using LazyColumn

Each counter includes:

“+” button to increase the count

“–” button to decrease the count

A name label (e.g., Counter_1, Counter_2, …)

All counters update independently

# Optional feature: 
Add and remove counters dynamically at runtime

# Tech Stack

Android (Kotlin)

Jetpack Compose

Material Design 3
